HTML Reference

Headings

Headings from 'h1' to 'h6' identify blocks and sections.

Paragraphs

Paragraphs of text go in 'p' tags

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in ul tags

Types of Lists:

  • Unordered lists ('ul')
  • Ordered Lists ('ol)

Ordered Lists

Ordered lists are numbered and go in 'ol' tags

Anchors (links)

Anchor Tags create links to other pages. The URL goes in the 'href' Go to DuckDuckGo to seacrch for information

Setting 'target="_blank"' will open the link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL goes in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump to the top of the page.

Text Emphasis

Italic

The 'em' element adds extra emphasis to the text within its tags. (Italics)

Bold

You can also use the 'strong' element to bold within the tags.
